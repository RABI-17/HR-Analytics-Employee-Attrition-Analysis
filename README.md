# HR-Analytics-Employee-Attrition-Analysis

## Overview
SQL-driven analysis of 1,470 IBM employees across 35 dimensions
to identify attrition drivers and build a predictive risk model.

## Live Dashboard
Tableau Public: (https://public.tableau.com/app/profile/rabinarayan.sahu/viz/IBMHRAttritionAnalysis_17812941961080/Dashboard1).

## Tools Used
- Python (Pandas, Seaborn, Matplotlib)
- SQL (SQLite — window functions, CTEs, CASE WHEN)
- Tableau Public
- Dataset: IBM HR Analytics (Kaggle)

## Key Findings
1. Overall attrition: [X]% — above industry benchmark of 10–15%
2. Overtime employees leave at 3x the rate of non-overtime employees
3. Sales department: [X]% attrition — highest in the company
4. Risk model: employees with 3+ risk factors have ~80% attrition rate
5. Manager stability is a hidden retention lever — longer tenure with
   same manager correlates strongly with lower attrition

## New Skills vs Previous Projects
- SQL window functions (RANK, AVG OVER PARTITION BY)
- CTEs for multi-step query logic
- Composite risk scoring model
- Dual-tool dashboard: Tableau Public + Power BI

## Notebook Structure
10-section report-style notebook with findings and
